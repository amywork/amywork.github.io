> SWAP 방식을 활용하여 무조건 큰 수에서 작은수를 빼는 코드를 작성해보았습니다.


### SWAP 방식을 활용하여 무조건 큰 수에서 작은수를 빼기

```swift
var number1: Int = 5
var number2: Int = 10

if number1 < number2
{
    var temp = number2
    number2 = number1
    number1 = temp
}

print(number1 - number2)
```
